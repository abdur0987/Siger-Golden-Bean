## Versi 1.2 — Fokus Kopi, Origin Lampung, Gallery, Proses Produksi, dan Legalitas

---

## 1. Ringkasan Proyek

Proyek ini adalah pengembangan **website landing page ringan, cepat, dan profesional** untuk brand ekspor kopi:

**Siger Golden Bean**  
**Tagline:** *Sumatran Premium Coffee Export*

Website ini ditujukan untuk membangun kepercayaan buyer internasional, terutama buyer dari **Mesir dan Timur Tengah**, dengan menampilkan identitas kopi Sumatra/Lampung, variasi produk kopi, asal daerah kopi, proses produksi, gallery, program sampel, legalitas, serta kontak cepat untuk inquiry.

Landing page harus dibuat dengan teknologi yang konsisten dengan proyek sebelumnya, yaitu pendekatan **Next.js + TypeScript + Tailwind CSS**, namun dibuat lebih sederhana dan ringan karena fungsi utamanya adalah promosi dan akuisisi buyer, bukan dashboard berat.

---

## 2. Tujuan Utama Website

1. Menampilkan brand **Siger Golden Bean** sebagai pemasok kopi Sumatra/Lampung yang premium dan siap ekspor.
2. Meyakinkan buyer bahwa produk memiliki asal yang jelas, proses produksi yang rapi, dan kualitas yang dapat divalidasi melalui sampel.
3. Memperkenalkan berbagai origin kopi Lampung secara kuat, bukan hanya satu jenis kopi umum.
4. Menyediakan informasi produk kopi yang mudah dibaca oleh importir, roastery, kafe, reseller, dan private label buyer.
5. Menyediakan area untuk menampilkan **sertifikat, legalitas, dan dokumen pendukung**.
6. Menampilkan gallery agar buyer melihat aktivitas, produk, kemasan, proses, dan bukti operasional.
7. Mengarahkan buyer untuk melakukan aksi cepat: **Request Sample**, **Get Catalog**, **Request Quotation**, atau **Chat via WhatsApp**.
8. Memastikan website cepat dibuka di desktop dan mobile dengan loading minimal.

### 7.5 Process Section — From Bean to Export
Tujuan: membuat buyer percaya bahwa produk tidak asal jual, tetapi melalui tahapan proses yang jelas.

Section ini wajib ada.

Judul:
**From Bean to Export: Our Coffee Preparation Process**

Tahapan yang ditampilkan:
1. **Sourcing from Origin**
   - Mengambil kopi dari area origin terpilih di Lampung dan Sumatra.
2. **Supplier & Farmer Selection**
   - Memilih supplier/petani berdasarkan kualitas, konsistensi, dan ketersediaan.
3. **Bean Selection**
   - Sortasi awal untuk memisahkan defect dan memastikan standar visual.
4. **Processing / Post-Harvest Check**
   - Menjelaskan natural, semi-washed, full-washed, honey, atau proses lain bila tersedia.
5. **Roasting Option**
   - Light-medium, medium, medium-dark, atau dark roast sesuai kebutuhan buyer.
6. **Grinding Option**
   - Whole bean atau ground; opsi fine grind untuk Arabic/Turkish style bila diminta.
7. **Packing**
   - Retail pack, sample pack, pouch, carton, atau bulk bag sesuai kebutuhan.
8. **Quality Check**
   - Cek aroma, roast date, packing, label, batch, dan sampel.
9. **Sample Dispatch**
   - Pengiriman sampel kecil untuk validasi buyer.
10. **Quotation & Order Confirmation**
   - Setelah buyer cocok, dibuat quotation dan skema order.
11. **Export Preparation**
   - Persiapan invoice, packing list, dokumen, dan koordinasi forwarder sesuai jenis pengiriman.

Catatan penting:
- Jangan klaim sertifikasi atau standar tertentu jika belum tersedia.
- Gunakan kalimat “available upon request” hanya untuk hal yang benar-benar bisa disediakan.

### 7.6 Quality Assurance Section
Tujuan: memperkuat kepercayaan buyer.

Isi:
- selected origin
- sample-first approach
- batch-based supply
- roast date control
- packing check
- buyer feedback loop
- documentation readiness

Contoh copy:
**We recommend a sample-first process before larger orders, so buyers can validate taste, aroma, grind size, roast profile, and packaging expectations.**

### 7.7 Gallery Section

Section ini wajib ada.

1. Coffee beans

5. Sample pack

7. Sorting process

Tujuan: menyediakan tempat untuk legalitas dan dokumen pendukung.

Section ini wajib ada walaupun dokumen belum lengkap. Jika dokumen belum tersedia, tampilkan sebagai “In Preparation” atau “Available Upon Request”, bukan klaim sudah ada.

Judul:

Kategori dokumen:
| Dokumen | Status Tampilan |

| NIB / Business Registration | Available / In Preparation |
| Halal Certificate | Available / In Preparation |
| PIRT / BPOM | Available / In Preparation |
| Company Profile | Available |
| Product Specification Sheet | Available |

| Phytosanitary Certificate | If required for green bean export |
| Fumigation Certificate | If required |

## 14. Performa dan Optimasi

Agar buyer nyaman, website harus ringan.

Aturan:
1. Jangan gunakan video autoplay di hero.
2. Jangan gunakan carousel berat.
3. Gunakan static rendering.
4. Gunakan gambar WebP/AVIF.
5. Gunakan `next/image`.
6. Lazy load gallery di bawah fold.
7. Gunakan icon ringan.
8. Tidak perlu API call saat first load.
9. Data produk/origin dari file lokal.
10. Font maksimal 1–2 family.
11. CSS sederhana dengan Tailwind.
12. Hindari animasi berlebihan.
13. Minimalkan JavaScript client-side.
14. Prioritaskan HTML statis dan server components.

Target:
- LCP cepat
- CTA terlihat langsung
- buyer memahami bisnis dalam 5–10 detik
- landing page nyaman di mobile

## 19. Catatan Strategis

Untuk tahap awal, landing page harus fokus pada **trust building** dan **buyer inquiry**, bukan transaksi langsung.

Strategi yang paling aman:
1. Tampilkan brand dan origin dengan kuat.
2. Tampilkan produk banyak, tetapi tetap rapi.
3. Tampilkan proses agar buyer yakin.
4. Tampilkan gallery sebagai bukti visual.
5. Tampilkan sertifikat/legalitas secara jujur.
6. Dorong buyer mulai dari sample kecil.
7. Jangan klaim kapasitas besar atau sertifikat yang belum tersedia.
8. Jangan terlalu banyak loading karena buyer internasional bisa langsung keluar jika website lambat.
