# Siger Golden Bean

Premium, lightweight landing page for Siger Golden Bean, a Sumatran coffee export brand focused on Lampung origins, buyer inquiry, sample requests, product catalog, and export document readiness.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Static local data files
- Optimized local WebP assets

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Validation

```bash
npm run lint
npm run build
```

## Notes

Contact details are centralized in `data/contact.ts`. Replace the placeholder WhatsApp and email values before publishing the site publicly.
